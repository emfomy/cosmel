Data Structure
==============

In ComsmEL, we define some data structure for collecting cosmetics database, blog articles and CKIP tools relatively.

Repository
----------

To maintain the cosmetics database from PIXNET in an organized way, we create a data structure named :class:`.Repo`, containing :class:`.Brand`, :class:`.Product`
